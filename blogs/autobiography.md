最近偶尔会想起很久之前的事导致失眠，看积极心理学说记录下来和分享出去比较好。应该一些事件都是私密的，不过基本不会有人看，也不介意有人看到。感觉我记忆力不是很好，也可能因为我很少仔细去回想，很多小时候的事的忘记了，或者记不清时间，比较近的事也会弄混时间。过去塑造出来的现在的我，现在的我也在一点点摆脱过去，记忆和过去都会自然地消失。我现在做的可以说是留下记忆，也可以说为了摆脱记忆。

# 出生 － 一般

我的记忆追溯不到这里。据说小时候有往亲戚的鞋里撒尿。家里条件不好还是年龄不够来着，看别人拿着一张纸去上学，路上捡了传单问父母，有这个能不能上学。因为没上过幼儿园，入学后成绩一直是倒数第一，得了倒数第二都很自豪。

一点都没有印象，很神奇我还有做过这样的事，可能对于小孩来说很一般。

# 小学 － 盗贼

偷东西是很不光彩的事，虽然小学发生的事非常多，可能偷东西这种事非常容易留在记忆里，所以记忆深刻。也可能是偷的太杂了，很多小孩偷过啤酒瓶、废品、家里的钱、同学的钱、小卖铺的东西、书店的书、自行车这类东西，而我都偷过。这些事一直在记忆中比价深刻，很长一段时间我去超市从兜里掏手机都觉得别扭，觉得容易产生误解，或许我还要赎罪很久。

应该在一二年级，我和女同桌关系很好，送过她各种小东西，感觉当时的我比现在都懂该怎么和异性相处。后排的同学比过一个像是小拇勾一起，大拇指对一起的手势调侃我们。忘了什么契机，得到了她家的电话，好像是作业的事打过一次，之后就再也没打过。不过号我居然现在还能背出来，可能是因为当了一段时间的密码的缘故。后面应该是换了一次座位就没有交流了，小时候的事好神奇。

三年级时刚开英语课，当时的英语老师很负责，因为我模拟家长签字的事家访过一次，这样的老师真少，可惜当时完全没想好好学英语。后面开始和交的朋友 A 偷小卖店和小超市的东西，有一次小卖店里貌似就我和 A 俩人台面上就剩下两三个三角形包装袋的糖，老板一转身，回过神来就少了一个，老板让我把糖放回去，上课铃也响了，没追究就让我走了。去过几次小超市，去买几毛钱的糖就偷出来几块钱的零食。之后和 A 闹了一次矛盾不在一起玩了，貌似没过多久 A 也转学了。

后面结交了朋友 B，当时小卖铺人太多了，圆牌会挂在低的架子上很容易扣下来往兜里塞，盗版的游戏王卡牌放在高的架子上倒是需要一些技术，想想可能有被发现了没有追究，因为每次 B 都会买一两盒。忘了什么原因 B 拿到过班里的钥匙几天，就和 B 从同学书包里偷钱，有几个同学家里条件好，书包里零钱多，少几块不会发现，也可能发现了没有找到我们头上。

还有次去书城偷过一本书，第一次成功带出去了，然后去楼上游戏厅逛了一圈，回去不知道该放哪就又带回去了，结果第二次带出来被发现了。最后把家长叫来，才解决的。当时我还着重强调书带出去过一次，他们可能理解成还偷了一本，后面解释清楚了是同一本。保安估计也是蒙的，心里可能在想：“这次不是被抓了么？非得说带出去过一次显得我可能工作不够认真？”当时真实各种蠢啊。

应该也是从三四年级就接触到游戏厅和索尼厅。现在再玩小时候玩的街机和索尼游戏，会发现和回忆里差的太多了，人类美化回忆的功能太强了。记忆比较深刻的是去游戏厅被老爸逮了用树条抽。中午会有各种理由早从家里出发去上学，说是去上学，其实都是找 B 索尼厅玩，反而玩到上课会迟到一会儿。应该没有闹过矛盾，好像他父母发现我是什么好小孩后，不太同意我们一起玩，也忘记什么时候开始和 B 的关系就淡了。

期间还有一段住托管班的经历，因为我妈当时在托管班打工。有次夜里看书，另外也住托管班的两个女生找我玩，三个人用军旗按类似麻将的方式玩，很欢乐。住托管班的两个女生有一个小我几届，父母应该很忙，五一托管班关闭，就把她交给我们家帮忙照看，但当时五一的印象只有会放《五一七天乐》电视节目。

五六年级接触到网络游戏。网络游戏确实对人影响很大，我仅有的三次自我毁灭行为，网游可能都算诱因之一，不过很幸运没有成功过。

第一次被朋友带入黑网吧玩的是一个回合制单机，后面入坑了梦幻西游。期间在黑网吧认识了朋友 C，可以认为他比我坏。当时他带着我偷了两次自行车。第一次在别人院子里，应该是挺久没人骑的。第二次就更坏了，是别人门口的，我们卖完回来路过那家门口，还看见丢车的人在焦急地寻找。这种经历我初中也有一次，那时我才真实感受到这种难受的情绪。后面还和 C 去抢过钱，貌似对方是他认识的人，搜了一下对方没钱，就结束了。

应该是六年级，也可能是初一，虽然现在想觉得六年级太早了，但根据回忆起的游戏内容推断应该是六年级，当时有记日记的习惯，好像前面还是写的游戏好友得了彩果（游戏道具）很替他开心，后面就写的我应该明天就不在了。可能当时本身我就容易上瘾和绝望，加上出去玩久了通常就挨训一起导致的。当时是割腕，幸运的是这样成功率不高。后面又记了几天，发现日记被我姐看了，她哭了。当时应该觉得后悔和气愤，后悔自己写了日记让本来过去的事又搞得别人伤心，气愤自己隐私被侵犯了，就烧了日记。现在应该还好，都过去这么久了，有人能看到当看故事就好，我自己看其实也像是故事。

应该是初中之前就不和 C 来往了，但是后面他会主动来欺负我，可能也是被他的朋友怂恿的。后面很正常进入了初中。

# 初中 － 坏小孩

校园暴力是存在的，至少在我身边存在。在我身上发生的只是被欺负过几回，欺负别人可能也是几回，虽然我只能记起一回。但个女生却一直遭受冷暴力，希望她已经走出来了。我和她可以说是没有交集，也应该下意识回避了本来该有的交集，因为和她有交集就会被议论。这样可能也算是施暴者一员。我现在知道这种事的一些原因，但仅仅是因为一些小事一开始被排挤，之后就愈演愈烈，难道只能自认倒霉么？可能没有答案。

初一总爱上课吃东西，同桌应该挺无语的。也挺爱出头，各种上课接话茬。初中两次“女装”的经历。一次是鞋坏了家里只有双带些粉色的鞋穿着上了几天学，当时觉得是女款，按现在来看没准是男款。一次是上补课班之前买了个外套，补课时一直穿着，回家脱下来仔细一看背面的英文 “fashion girl”，太逗了。

初二应该是最波折的阶段。看来管制容易伤害别人和伤害自己的东西是好的。当时依旧沉迷于游戏，应该是一次偷偷出去玩挨训后，触发了绝望，在手上划了几刀（当时觉得手腕容易太明显不好解释，手上可能好解释为不小心的）喝了之前买好的医用酒精。当时买了两瓶第一瓶喝完，嗓子都噎住了，第二瓶应该就没怎么喝下。本来当挂定了，结果居然正常按时醒来了，醒来后很恶心，去门口忘记有没有吐出来，但当天是上学的日子，洗了把脸就去上学了。同桌看到我手上的伤很关心，我也依照剧本说不小心的，不知道有没有被识破。真幸运当时我只能搞到医用酒精，并且天真地认为这东西能致死。

后面应该还有一次挨训，这次应该一开始就没有觉得会成功。之前有手腕上伤口很深也没事的经验，这次是尽量找准把买时脉搏的位置，很明显也没有成功。这是我最后一次自我毁灭的经历，后面就有些转变了。

初三下就比较正常了，好好学习了一段时间。参加中考，擦边过了第一批次高中线。当时开始了一种叫分招的政策，过了第一批次线可能能进入重点高中，不过我差几名没进去。当时还有想上职高的想法被否定了，没准当时上职高学计算机又是另一种人生。后面问了好一些的高中，可能交上几万可以上，家里看我不是特别想上也就没这样操作。可能也是好事，也许在普通高中才能有如此强烈的正反馈，体验如此复杂的情绪。可能也不太普通，高中军校据说全国只有两所，而且我去的那所很小，小到只有几个班。

暑假想去打工，去了一家自助餐店，算是保洁和服务员，上午擦椅子，中午服务客人，挺累但我觉得坚持下来还是没问题的，直到下午。本来以为下午会继续做普通的工作或者清闲一些，但是管理拿来了传单，发传单的活我干不了，只是尝试了发第一张，对方摆手拒绝，我就放弃了。后面去亲戚家的饭店打工上茶、点菜、传菜和打杂，这种工作真还挺充实，也不会像现在的工作各种压力。打完工回来后买了《C 语言程序设计教程》准备学一学。

# 高中 － 疯狂

高中一开始我可能变成了大家眼中的令人羡慕的优等生，学习好，体育好。但随着时间快速推进或许发生了一些变化。高中疯狂学习、运动、玩游戏和做另类的事。像是偶然，实际可能是必然，高中最后也是另类结局。

高一和高二前半段是最高光的时刻，因为是普通高中学习排名很高，有过一段时间是最高。体育也很好，加入了体育队，长距离跑步名次最好。受到正反馈的我开始疯狂学习和运动，上下学路上、体育队训练间隙都在背单词。寒暑假也是一直学习，觉得白天效率低就学一个通宵，白天睡觉。这段时间玩游戏很少，不过高二上的寒假买了地下城与勇士的年套和一个 70 级的男大枪号，当然是用助学金。

高二还有一些插曲，一开始我被安排讲台旁边的在第零排（第一排前面的那一排），并且不会每周轮换，这是成绩好的学生的特权，同坐是个比较活泼爱玩的女生。但我受不了第一排，一部分因为当时我认为第一排会很受关注，我不太想受到关注。我将这个想法和同桌说了，准备后面换到最后一排，并且说明了这是我个人原因。她可能有些生气，因为老师可能会有别的想法，后面她逃课和另一个同学打球，回来后同意了。后面我找老师换到最后一排，也是固定座位，不会轮换。

也找老师申请过想要跳级，不过没有成功。老师让我做一张重点高中的高二试卷，确实难很多不会做，现在来看不确定做的很好会不会有改变。

当时还买了特别厚的那本蓝皮的《C Primer Plus》，囫囵吞枣地看完了。高考不是数学有一道流程图题嘛，本来以为这道题我擅长绝对稳了，结果最后对答案发现做错了。

高二后面开始沉迷英雄联盟，现在看来就是人菜瘾大。当时有碰巧父母去外地干活，高中减负，这一下真是玩疯了。每天放学比初中生都早，放学就去网吧，玩上头了就通宵，五一放了七天假，连玩了七天。当时用的父母的身份证，后面他们回来拿身份证要用发现不在家里，最后是从学校找到我要到的身份证。后面没有身份证只能去黑网吧或者能直接上机的网吧，这样的网吧很多。

高三压力挺大，可能也是青春期的缘故，发现了自残可以解压，并且几乎没什么影响就有些滥用了。当时算是暗恋班花，但知道不是一个世界的人。周六放学会绕远到河边的路回家，也有一部原因是知道她家在这个方向，期待着什么吧。回家还是希望努力学习，但周六各种心情混杂，有时会直接来一刀让自己好受一些。

当时我认为自学比上课效率高，高三下拒绝了尖子生的小班课。一模成绩很低，因为考完才理解到我们学校普通，之前的考试和模拟考不是一个段位。之后申请休假在家自学，各种争论后没有成功。依旧是古老的解压手段，不过这次没做好准备，门没有反锁被发现了。这可能是父母第一次发现我有自残行为，也可能不是，他们反应激烈，但没我想象中的激烈，可能他们知道这种伤口问题不大。清洗好伤口，用纸巾加上胶带盖住防止弄脏衣服和床单被罩就好。睡觉，第二正常去上学。

后面我姐去上学后，又和父母商量，这次成功了，去学校请了一个月假在家自学到二模。结果二模成绩更低，虽然我知道是因为当时没复习完。后面正常去学校，正常的上课，正常上小班课。

很快到了三模，考理综迟到了，成绩更低，可能因为这是最后一次没给排名，按分数班级的中游可能都够不上。感觉这时很多人可能会觉得本来很好的孩子高三不行了。后面又上了一段时间课开始考前放假，我们那次假好长啊，好像有七天还是十天来着。放假当天去通宵玩游戏。然后就没有出门在家除了正常的复习，还将所有课本画的重点抄了一遍，我觉得已经准备好了。

最后的高考结果还行，一本线 501，我超了两分 503，又是擦边过了线。我们是军校高中，只要过了一本线且军检通过可以上很好的军事相关的大学。我们那届有四人过了一本线，据说只有我没有过军检。大概率是手腕的伤疤导致的，军检还是很严格的。不过一般和人讲起这件事我都说可能体重或视力不达标吧。

暑假还是去亲戚的饭店打工，和初中毕业类似，报志愿也是去妈妈的朋友家报，她家有电脑和网络。本来想去远一些的地方，最后比较随意地报了辽宁的大学，专业本来选的物理，调剂到了地理信息科学。至于为什么没有报我一直喜欢的计算机专业，原因之一是计算机专业贵一些，尤其是大三和大四要比其他专业贵好多，之二是我觉得专业无所谓，主要看自己想做什么。
