简单来看可以直接选 exFAT，因为这个文件系统可以在 Windows、MacOS、Linux 上读写。之前我一直用 Window 所以文件系统选择的 NTFS，但是后面换成用 MacOS 发现默认只能读无法写，所以就换成 exFAT 了。
