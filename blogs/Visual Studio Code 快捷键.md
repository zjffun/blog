# 代码提示（自动补全，自动完成）

默认是快捷键是`Ctrl+Space`，与搜狗输入法切换中英文的快捷键冲突了。。可以改搜狗输入法的快捷键（`Shift`切换中英文就完全够用），或是将 VSCode 的`editor.action.triggerSuggest`快捷键改掉（改成`Alt+/`也不错）。
